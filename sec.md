# SEC 文件種類詳細分類（公開上市公司適用）

本分類依照美國證券交易委員會 (SEC) 的規定，列出公開上市公司必須提交的所有主要文件種類。下列分類分為「登記與發行文件」、「定期報告文件」、「內部人持股及交易文件」、「股東投票與公司治理文件」、「收購要約與併購相關文件」、「投資公司專用文件」以及「其他雜項文件」。文件修正版本均以「/A」標示，表示為原始文件的修正或補充說明版本。

 

## 1. 登記與發行文件  
（依據《1933 證券法》及相關規定，用於證券發行登記與募集說明）

### 1.1 基於《1933 證券法》的 S 系列登記文件  
- **Form S-1**：首次公開發行（IPO）登記聲明  
  - **Form S-1/A**：Form S-1 的修正文件  
- **Form S-2**：適用於公司具備一定運作歷史之登記聲明  
- **Form S-3**：成熟公司使用之簡化登記聲明  
  - **Form S-3/A**：Form S-3 的修正文件  
- **Form S-4**：用於併購、交換報價及其他商業組合交易之登記聲明  
  - **Form S-4/A**：Form S-4 的修正文件  
- **Form S-6**：單位投資信託登記聲明  
- **Form S-8**：發行給員工（員工股票計劃等）的登記聲明  
- **Form S-11**：房地產投資信託（REIT）或房地產企業登記聲明  
  - **Form S-11/A**：Form S-11 的修正文件

### 1.2 外國私人發行的 F 系列文件  
- **Form F-1**：外國私人發行人首次登記聲明  
  - **Form F-1/A**：Form F-1 的修正文件  
- **Form F-3**：外國成熟發行人使用之簡化登記聲明  
  - **Form F-3/A**：Form F-3 的修正文件  
- **Form F-4**：外國私人發行人在併購交易中使用之登記聲明  
  - **Form F-4/A**：Form F-4 的修正文件  
- **Form F-6**：針對美國存托憑證 (ADR) 的登記聲明  
  - **Form F-6/A**：Form F-6 的修正文件  
  - **Form F-6EF**：自動生效版本之 Form F-6  
- **Form F-7**：加拿大發行人以現金認購方式使用之登記聲明  
  - **Form F-7/A**：Form F-7 的修正文件  
- **Form F-8**：加拿大發行人用於交換或併購交易之登記聲明  
  - **Form F-8/A**：Form F-8 的修正文件  
- **Form F-9**：針對特定加拿大投資等級債券或優先股之登記聲明  
  - **Form F-9/A**：Form F-9 的修正文件  
- **Form F-10**：加拿大發行人其他登記聲明  
- **Form F-80**：加拿大發行人交換性質之登記文件

### 1.3 Regulation A 文件  
- **Form 1-A**：根據 Regulation A 進行公開發行所用之募集說明書  
  - **Form 1-A/A**：Form 1-A 的修正文件  
- **Form 1-A POS**：Form 1-A 發生後修正文件  
- **Form 1-A-W**：撤回 Form 1-A 之聲明文件  
  - **Form 1-A-W/A**：Form 1-A-W 的修正文件

 

## 2. 定期報告文件  
（用以持續提供公司財務、業務及風險資訊）

### 2.1 年報文件  
- **Form 10-K**：美國公司年度報告，包括業務描述、審計後財務報表、風險因素、管理層討論與分析 (MD&A) 等  
  - **Form 10-K/A**：Form 10-K 的修正文件  
  - （另有變體如 Form 10-K405 與 Form 10-K405/A）  
- **Form 20-F**：外國私人發行人年度及過渡報告  
  - **Form 20-F/A**：Form 20-F 的修正文件  
- **Form 40-F**：適用於部分加拿大上市公司之年度報告  
  - **Form 40-F/A**：Form 40-F 的修正文件  
- 投資公司專用報告（分別列出下列各項）：
  - **NSAR-A**：投資公司半年度報告  
  - **NSAR-B**：投資公司年度報告  
  - **NSAR-BT**：投資公司過渡性年度報告  
  - **NSAR-U**：投資公司針對單位投資信託之年度報告

### 2.2 季報文件  
- **Form 10-Q**：美國公司季度報告，包含未經審計財務資料  
  - **Form 10-Q/A**：Form 10-Q 的修正文件  
- **Form 10-KT**：與年度報告相關之過渡性報告  
  - **Form 10-KT/A**：Form 10-KT 的修正文件  
- **Form 10-QT**：季度報告之過渡性版本  
  - **Form 10-QT/A**：Form 10-QT 的修正文件  
- **Form 10-QSB**：供小型企業使用之簡化季度報告  
  - **Form 10-QSB/A**：Form 10-QSB 的修正文件

### 2.3 即時（事件）報告文件  
- **Form 8-K**：披露重大事件或變動之即時報告  
  - **Form 8-K/A**：Form 8-K 的修正文件  
- **Form 6-K**：外國發行人披露重大事項之即時報告  
  - **Form 6-K/A**：Form 6-K 的修正文件

 

## 3. 內部人持股及交易文件  

### 3.1 內部人披露文件  
- **Form 3**：首次向 SEC 披露內部人（董事、主管等）持股情形  
- **Form 4**：內部人持股或交易變動報告  
  - 如有修正，可能標示為 4/A  
- **Form 5**：內部人年度持股交易報告  
  - **Form 5/A**：Form 5 的修正文件

### 3.2 機構及大額投資者披露文件  
- **Form 13F-HR**：由機構投資經理每季提交之持股報告  
  - **Form 13F-HR/A**：Form 13F-HR 的修正文件  
- **Form 13F-NT**：機構投資者的持股通知報告  
  - **Form 13F-NT/A**：Form 13F-NT 的修正文件  
- **Form 13FCONP**：機構投資者的另一版本持股報告  
- **Form 13H** 系列（分別列出以下各項）：
  - **Form 13H**：大額投資者或大額交易者的初始報告  
  - **Form 13H-Q**：大額投資者季度報告  
  - **Form 13H-A**：大額投資者年報  
  - **Form 13H-I**：大額投資者狀態報告（失效或暫停時）  
  - **Form 13H-R**：大額投資者重新激活報告  
  - **Form 13H-T**：大額投資者終止報告

 

## 4. 股東投票與公司治理文件  

### 4.1 正式代理書與資訊披露文件  
- **Form DEF 14A**：正式代理書（Definitive Proxy Statement），提供股東投票資訊  
- **Form DEF 14C**：正式資訊說明文件（Definitive Information Statement），用於不涉及代理投票的情形  
- **Form DEFA14A**：由非管理層提供的補充代理材料  
- **Form DEFA14C**：由非管理層提供的補充資訊說明材料  
- **Form DEFC14A**：針對爭議性或對抗性情形而提交的正式代理書  
- **Form DEFC14C**：針對爭議性或對抗性情形而提交的正式資訊說明文件  
- **Form DEFM14A**：用於併購或重大交易時的正式代理書  
- **Form DEFM14C**：用於併購或重大交易時的正式資訊說明文件  
- **Form DEFN14A**：由非管理層內部人提交的正式代理書  
- **Form DEFR14A**：經修正的正式代理徵求材料  
- **Form DEFR14C**：經修正的正式資訊說明文件

### 4.2 初步與修正型代理文件  
- **Form PRE 14A**：初步代理書  
- **Form PRE 14C**：初步資訊說明文件  
- **Form PREC14A**：與爭議性徵求相關之初步代理書  
- **Form PREC14C**：與爭議性資訊披露相關之初步說明文件  
- **Form PREM14A**：與併購或重大交易相關之初步代理書  
- **Form PREM14C**：與併購或重大交易相關之初步資訊說明文件  
- **Form PREN14A**：由非管理層提交之初步代理書  
- **Form PRER14A**：初步修正代理徵求材料  
- **Form PRER14C**：初步修正資訊說明文件  
- **Form PRRN14A**：經修正的初步代理書（由非管理層提交）

 

## 5. 收購要約與併購相關文件  

### 5.1 收購披露文件  
- **Schedule 13D**：當公司股權超過 5% 時必須提交的收購說明文件  
  - **Schedule 13D/A**：Schedule 13D 的修正文件  
- **Schedule 13G**：適用於被動投資者的簡化收購披露表格  
  - **Schedule 13G/A**：Schedule 13G 的修正文件

### 5.2 收購要約與併購文件  
- **Form S-4**：用於併購、交換報價與業務組合交易的登記聲明  
  - **Form S-4/A**：Form S-4 的修正文件  
  - **Form S-4MEF**：依據 Rule 462(b) 補充發生後的新登記聲明（若適用）  
- **Schedule TO-T**：第三方收購要約時提交的收購要約文件  
- **Form SC 14D9**：收購邀約徵求說明文件  
  - **Form SC 14D9/A**：Form SC 14D9 的修正文件  
- **Form SC 14F1**：關於董事多數變動情形之說明文件  
  - **Form SC 14F1/A**：Form SC 14F1 的修正文件

 

## 6. 投資公司專用文件  

### 6.1 投資公司登記文件  
- **Form N-1A**：開放式管理投資公司（共同基金）的登記聲明  
  - **Form N-1A/A**：Form N-1A 的修正文件  
- **Form N-2**：封閉式投資公司的登記聲明  
  - **Form N-2/A**：Form N-2 的修正文件  
- **Form N-3**：管理投資公司獨立賬戶登記聲明  
  - **Form N-3/A**：Form N-3 的修正文件  
- **Form N-4**：單位投資信託登記聲明  
  - **Form N-4/A**：Form N-4 的修正文件  
- **Form N-5**：小型企業投資公司登記聲明  
- **Form N-14**：投資公司於併購或業務組合交易時提交的登記聲明  
  - **Form N-14**（原始文件）  
  - **Form N-14/A**（修正文件）  
  - **Form N-14 8C**（適用於封閉式投資公司於併購交易中）  
  - **Form N-14 8C/A**（修正 N-14 8C）  
  - **Form N-14AE**（另一變體）  
  - **Form N-14AE/A**（修正 N-14AE）  
- **Form 485APOS**：投資公司登記聲明之發生後修正文件（Post-effective Amendment）  
- **Form 485BPOS**：投資公司年度更新之發生後修正文件

### 6.2 投資公司定期與代理文件  
- **Form N-CSR**：經認證的年度股東報告  
  - **Form N-CSR/A**：N-CSR 的修正文件  
- **Form N-CSRS**：經認證的半年度股東報告  
  - **Form N-CSRS/A**：N-CSRS 的修正文件  
- **Form N-PX**：管理投資公司代理投票記錄年度報告  
  - **Form N-PX/A**：N-PX 的修正文件  
- **Form N-Q**：季度投資組合持股報告  
  - **Form N-Q/A**：N-Q 的修正文件  
- **NSAR 系列文件**：  
  - **NSAR-A**：半年度管理公司報告  
  - **NSAR-B**：年度管理公司報告  
  - **NSAR-BT**：過渡性年度報告  
  - **NSAR-U**：單位投資信託年度報告

 

## 7. 其他雜項文件  

### 7.1 修正、終止與撤回文件  
- 所有文件如有修正版本均標示「/A」，例如：Form 10-K/A、Form 8-K/A、Form S-1/A。  
- **Form 15-12B**：依據第 12(b) 條終止某一證券登記的通知  
  - **Form 15-12B/A**：15-12B 的修正文件  
- **Form 15-12G**：依據第 12(g) 條終止某一證券登記的通知  
  - **Form 15-12G/A**：15-12G 的修正文件  
- **Form 15-15D**：依據第 13 及 15(d) 條暫停提交報告義務的通知  
  - **Form 15-15D/A**：15-15D 的修正文件  
- 針對外國發行人之文件：
  - **Form 15F-12B**、**Form 15F-12B/A**、**Form 15F-12G**、**Form 15F-12G/A**、**Form 15F-15D**、**Form 15F-15D/A**

### 7.2 Rule 144 出售通知文件  
- **Form 144**：內部人或控股股東依據 Rule 144 出售限制性證券時提交之通知  
  - **Form 144/A**：Form 144 的修正文件

### 7.3 發生後修正與其他特定用途文件  
- **POS 系列文件**（發生後修正文件）：  
  - **POS 8C**：閉式投資公司發生後修正文件  
  - **POS AM**：提交時未立即生效之發生後修正文件  
  - **POS AMI**：適用於僅依據 1940 法案提交的發生後修正文件  
  - **POSASR**：針對自動架上登記聲明之發生後修正文件（用於 Form S-3ASR 或 Form F-3ASR）  
  - **POS462B**：根據 Rule 462(b) 提交之發生後修正文件  
  - **POS462C**：根據 Rule 462(c) 提交之發生後修正文件  
- **CORRESP**：與 SEC 溝通所使用的函件文件  
- 特定用途文件：  
  - **Form 10-D**：針對資產擔保發行人提交之文件  
  - **Form 1-E**、**Form 1-E AD**、**Form 2-E**：針對銷售材料提交之文件  
- 撤回或撤銷文件：  
  - **Form AW**：申請撤回先前提交的文件  
  - **Form AW WD**：撤回申請文件  
  - **Form RW**：登記撤回申請  
  - **Form RW WD**：撤回登記撤回申請

> **補充說明：**  
> 1. 本分類針對公開上市公司所必須提交之文件，列出所有主要文件種類和其修正版本，絕不以「等」、「等等」方式省略。  
> 2. 文件中每個項目均單獨列出，修正版本以「/A」標示。  
> 3. SEC EDGAR 系統所涵蓋的文件種類數量眾多，此分類基於常見及主要使用的文件作整理，供參考使用。
