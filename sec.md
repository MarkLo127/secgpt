# SEC 文件種類詳細分類（公開上市公司適用）

本分類全面整理了公開上市公司依據美國證券交易委員會 (SEC) 規定所必須提交的各類文件。文件分為登記與發行、定期報告、內部人持股與交易、股東投票與公司治理、收購要約與併購、投資公司專用文件，以及其他雜項文件。各文件若附有「/A」後綴，表示為原始文件的修正或補充說明版本。

 

## 1. 登記與發行文件  
（依據《1933 證券法》及相關規定，主要用於證券發行登記及募集說明）

### 1.1 基於《1933 證券法》的登記文件  
- **S 系列文件**  
  - **Form S-1**：首次公開發行（IPO）登記聲明  
    - 修正版本：S-1/A  
  - **Form S-2**：適用於公司具備一定歷史（但不一定符合 S-3 要求）的登記聲明  
  - **Form S-3**：成熟公司使用的簡化登記聲明  
    - 修正版本：S-3/A  
  - **Form S-4**：用於併購、交換報價及其他商業組合交易的登記聲明  
    - 修正版本：S-4/A  
  - **Form S-6**：單位投資信託登記聲明  
  - **Form S-8**：發行給員工（例如員工股權計劃）的登記聲明  
  - **Form S-11**：專用於房地產投資信託（REIT）或房地產類公司的登記聲明  
    - 修正版本：S-11/A

### 1.2 外國私人發行（F 系列）文件  
- **Form F-1**：外國私人發行人首次登記聲明  
  - 修正版本：F-1/A  
- **Form F-3**：外國成熟發行人使用的簡化登記聲明  
  - 修正版本：F-3/A  
- **Form F-4**：外國私人發行人在併購交易中使用的登記聲明  
  - 修正版本：F-4/A  
- **Form F-6**：針對美國存托憑證（ADR）的登記聲明  
  - 修正版本：F-6/A  
  - 另有自動生效版本：F-6EF  
- **Form F-7**：加拿大發行人以現金認購方式的登記聲明  
  - 修正版本：F-7/A  
- **Form F-8**：加拿大發行人用於交換或併購交易的登記聲明  
  - 修正版本：F-8/A  
- **Form F-9**：針對特定加拿大投資等級債券或優先股的登記聲明  
  - 修正版本：F-9/A  
- **Form F-10** 及 **Form F-80**：其他針對加拿大發行人的登記文件  
  - （若有修正版本，則以相同規則標示，例如 F-10/A 等）

### 1.3 Regulation A 文件  
- **Form 1-A**：根據 Regulation A 發行的募集說明書  
  - 修正版本：1-A/A  
- **Form 1-A POS**：1-A 發生後修正（Post-effective Amendment）  
- **Form 1-A-W**：1-A 的撤回聲明  
  - 修正版本：1-A-W/A

 

## 2. 定期報告文件  
（持續向 SEC 提交更新資料，使投資人能隨時掌握公司最新狀況）

### 2.1 年報文件  
- **Form 10-K**：美國公司年度報告，包含業務概況、審計財務報表、風險因素、管理層討論與分析 (MD&A) 等  
  - 修正版本：10-K/A  
  - 其他變體（如 10-K405 與其修正版本）  
- **Form 20-F**：外國私人發行人的年度及過渡報告  
  - 修正版本：20-F/A  
- **Form 40-F**：適用於部分加拿大上市公司的年度報告  
  - 修正版本：40-F/A  
- 投資公司專用的**NSAR 系列**：例如 NSAR-A（半年度報告）、NSAR-B（年度報告）、NSAR-BT（過渡年報）、NSAR-U（針對單位投資信託）

### 2.2 季報文件  
- **Form 10-Q**：美國公司季度報告（未經審計，披露季度業績）  
  - 修正版本：10-Q/A  
- **Form 10-KT / 10-KT/A**：年度報告的過渡性報告及修正  
- **Form 10-QT / 10-QT/A**：季度報告的過渡性版本及修正  
- 小型企業可使用**Form 10-QSB**（及其修正版本 10-QSB/A）  

### 2.3 即時（事件）報告  
- **Form 8-K**：披露重大事件、變動或其他臨時資訊  
  - 修正版本：8-K/A  
- **Form 6-K**：外國發行人用以報告重要事件的即時報告  
  - 修正版本：6-K/A

 

## 3. 內部人持股及交易文件  
（用於披露公司內部高階管理人員、董事及大股東的持股變化以及交易活動）

### 3.1 內部人披露文件  
- **Form 3**：首次披露內部人（董事、高階管理人）的持股狀況  
- **Form 4**：內部人持股變動報告  
  - 修正版本：4/A（若適用）  
- **Form 5**：內部人年度持股變動報告  
  - 修正版本：5/A

### 3.2 機構與大額投資者披露  
- **Form 13F 系列**  
  - **13F-HR**：機構投資者持股報告（持有 10-K 表格中的證券）  
    - 修正版本：13F-HR/A  
  - **13F-NT**：機構投資者持股通知  
    - 修正版本：13F-NT/A  
  - **13FCONP**：另一版本的機構持股報告  
- **Form 13H 系列**  
  - 包括 13H、13H-Q、13H-A、13H-I、13H-R、13H-T  
    - 用於大額交易者及必須披露交易詳情的投資者

 

## 4. 股東投票與公司治理文件  
（用於召開股東會議、投票及披露公司治理與管理層相關資訊）

### 4.1 正式代理書與資訊披露  
- **Form DEF 14A**：正式（Definitive）代理書，提供股東投票所需詳盡資訊  
- **Form DEF 14C**：正式資訊說明書（適用於非委任投票情境）  
- **其他相關文件**：  
  - **DEFA14A**：非管理層提供的附加代理材料  
  - **DEFA14C**：非管理層附加資訊說明材料  
  - **DEFC14A**、**DEFC14C**：針對爭議性或對抗性投票情形的正式代理與資訊文件  
  - **DEFM14A**、**DEFM14C**：併購交易中使用的正式代理與資訊文件  
  - **DEFN14A**：非管理層正式代理文件  
  - **DEFR14A**、**DEFR14C**：修正型正式代理或資訊文件

### 4.2 初步與修正型代理文件  
- Preliminary / Revised 代理材料：  
  - **PRE 14A**、**PRE 14C**、**PREC14A**、**PREC14C**  
  - **PREM14A**、**PREM14C**、**PREN14A**  
  - **PRER14A**、**PRER14C**、**PRRN14A**

 

## 5. 收購要約與併購相關文件  
（用於披露收購、併購及相關重大交易資訊）

### 5.1 收購披露  
- **Schedule 13D**：當持有某公司 5% 以上投票權時必須披露的收購說明文件  
  - 修正版本：13D/A
- **Schedule 13G**：適用於被動投資者的簡化收購披露表格  
  - 修正版本：13G/A

### 5.2 併購及收購相關的其他文件  
- **Tender Offer Filings**：  
  - 例如：SC 14D9、SC 14D9/A（收購邀約說明）  
  - 及第三方收購要約文件，如 **Schedule TO-T**（Tender Offer - Third Party）  
- **Form S-4** 及其變體：用於併購與交換報價的登記文件  
  - 修正版本：S-4/A、S-4MEF

 

## 6. 投資公司專用文件  
（針對管理投資公司、共同基金、封閉式基金等的專用登記與報告文件）

### 6.1 投資公司登記文件  
- **Form N-1A**：開放式管理投資公司（共同基金）的登記聲明  
  - 修正版本：N-1A/A
- **Form N-2**：封閉式投資公司的登記文件  
  - 修正版本：N-2/A
- **Form N-3**：管理投資公司獨立賬戶登記聲明  
  - 修正版本：N-3/A
- **Form N-4**：單位投資信託登記聲明  
  - 修正版本：N-4/A
- **Form N-5**：小型企業投資公司登記聲明  
- **Form N-14**：投資公司在併購或組合交易時使用的登記聲明  
  - 亦有變體如 N-14 8C、N-14 8C/A、N-14AE、N-14AE/A 等
- **Forms 485APOS / 485BPOS**：投資公司登記聲明之發生後修正

### 6.2 投資公司股東與定期報告  
- **Form N-CSR**：經認證的年度股東報告  
  - 修正版本：N-CSR/A  
- **Form N-CSRS**：經認證的半年度股東報告  
  - 修正版本：N-CSRS/A  
- **Form N-PX**：年度代理投票記錄報告  
  - 修正版本：N-PX/A
- **Form N-Q**：季度投資組合持股報告  
  - 修正版本：N-Q/A
- 以及 **NSAR 系列**：如 NSAR-A、NSAR-B、NSAR-BT、NSAR-U 等

 

## 7. 其他雜項文件  
（包含修正、通知、撤回及其他特定用途的文件）

### 7.1 修正、通知與撤回相關文件  
- 各類文件的**修正版本**均以「/A」標示（例如：10-K/A、8-K/A、S-1/A 等）  
- **終止登記或暫停報告通知**：  
  - 例如：15-12B、15-12B/A、15-12G、15-12G/A、15-15D、15-15D/A  
  - 及針對外國發行人終止報告義務：15F-12B、15F-12B/A、15F-12G、15F-12G/A、15F-15D、15F-15D/A

### 7.2 Rule 144 及其他銷售通知  
- **Form 144**：內部人或控股股東依 Rule 144 出售限制性證券時的申報文件  
  - 修正版本：144/A

### 7.3 發生後修正及其他特定用途文件  
- **POS 系列文件**：  
  - 如 **POS 8C**（閉式投資公司發生後修正）、**POS AM**、**POS AMI**、**POSASR**、**POS462B**、**POS462C**  
- **CORRESP**：與 SEC 溝通所用的函件或說明文件  
- 其他特定用途文件：  
  - 如針對資產擔保發行人的 **Form 10-D**  
  - 針對銷售材料的 **Form 1-E**、**Form 1-E AD**、**Form 2-E** 等

> **補充說明：**  
> 1. 本分類僅涵蓋公開上市公司必須提交的文件；針對私募或非公開發行（例如 Form D）的文件不包含在內。  
> 2. 大部分文件如有修正版本均以「/A」標示。  
> 3. SEC EDGAR 系統累積的文件種類眾多，以上分類整合了常見及主要類型，作為參考依據。
